Project Description

The project is a simple, smart system for predicting early fainting or falling using a wearable device (a small bracelet/unit) equipped with biometric sensors such as a pulse sensor and a blood oxygen sensor (SpO2 sensor).
The device continuously collects the user's biometric data, and whenever a sudden drop or abnormal change in these indicators is detected, the system predicts the possibility of fainting or falling.

Once this occurs, the system activates an alert mechanism through several channels:

Alerts the user directly via a sound or vibration on the device.

Sends an SMS message to relatives or other people close to them to inform them of their condition.

Notification via a simple mobile/web app that displays the user's status and previous alerts.
